# C++ Calculator

## A Command-Line Calculator Built in C++

</br>

### Input the equation required following the below rules&#58;

- Space-seperate all items
- Use ASCII symbols for operators

</br>

### Example&#58;

    10 + 20 - 5

### Would return&#58;

    25

</br>

### The avaible operations are currently the below&#58;

- `+` for addition
- `-` for subtraction
- `*` for multiplication
- `/` for division
- `^` for powers

</br>

### Or &#58;

    4 ^ 2 + 3

### Will return&#58;

    19
