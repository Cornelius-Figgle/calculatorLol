
# C++ Calculator

A Command-Line Calculator Built in C++

## Usage

- Follow prompts
- Space-seperate all items
- Use ASCII symbols for operators

```bash
>>> 10 + 20 - 5

25
```

### The avaible operators are currently the below&#58;

```bash
- `+` for addition
- `-` for subtraction
- `*` for multiplication
- `/` for division
- `^` for powers
```
